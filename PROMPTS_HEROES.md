# Prompts IA pour les 22 sprites de héros

Document à fournir à Gemini / Imagen / ChatGPT / Midjourney / autre modèle de génération d'image.

## ⚠️ Consignes communes à TOUS les prompts

Pour obtenir une cohérence visuelle avec les 4 sprites existants (warrior.png, archer.png, mage.png, healer.png) et entre les 22 héros :

- **Style** : 2D pixel art RPG sprite, retro video game style, reminiscent of Final Fantasy Tactics / Disgaea / Octopath Traveler
- **Format** : PNG, 128×128 px, transparent background (no border, no frame, no background scenery)
- **View** : full body, front-facing (3/4 view acceptable), standing idle pose
- **Lighting** : soft top-down lighting, slight drop shadow beneath the feet
- **Outline** : thin 1px dark outline around the character
- **Color palette** : vibrant but not saturated, slight cell-shading
- **No text, no logo, no watermark, no border/frame**

Chaque prompt est déjà formulé en anglais et prêt à être copié-collé.

## 📁 Où placer les fichiers générés

Une fois les PNG reçus, les placer dans :

```
C:\Idle game\assets\sprites\allies\heroes\<hero_id>.png
```

Exemple : `assets/sprites/allies/heroes/garde_royal.png`

# 📌 Conseils pour l'utilisation

## Si tu utilises **Midjourney** ou **Stable Diffusion**

Ajouter à la fin de chaque prompt :
```
--style raw --ar 1:1 --v 6.1 --quality 2
```

## Si tu utilises **Gemini / Imagen**

Les prompts sont déjà formulés en langage naturel, juste les copier-coller tels quels. Préciser à la fin :
```
Output: 128x128 PNG, transparent background, no frame or border.
```

## Si tu utilises **ChatGPT avec DALL-E**

Même principe, copier-coller. Au besoin, ajouter :
```
Generate as a PNG with transparent background, 128x128 pixels, no background scenery, no frame.
```

## Retouche après génération

Les modèles IA produisent parfois :
- Un fond qui n'est pas transparent → utiliser **remove.bg** gratuit pour rendre le fond transparent
- Une taille > 128×128 → redimensionner avec **Photopea** (en ligne, gratuit) avec interpolation "nearest neighbor" pour préserver le pixel art
- Un style trop "réaliste" → régénérer en ajoutant `16-bit SNES era` ou `clearly pixelated` dans le prompt

Une fois les 22 PNG récupérés, les déposer dans `assets/sprites/allies/heroes/` et me dire "c'est bon" — je ferai la passe de code pour les charger et les assigner à chaque héros en combat.
